## 完整的 7 文件结构

### 文件 ①: IdentityCard.h

**位置**: `Framework.edu/IdentityCard/IdentityCard.h`

**作用**: 声明框架依赖关系

**关键规则**:
- ❌ 不要使用 `#ifndef` 守卫
- ✅ 使用 `AddPrereqComponent` 声明依赖

**模板**:
```cpp
// IdentityCard.h
// Framework: <FrameworkName>

AddPrereqComponent("System", Public);
AddPrereqComponent("ObjectModelerBase", Public);
```

**常见依赖**:
- `System` - 总是需要
- `ObjectModelerBase` - 组件系统
- `Mathematics` - 数学运算
- `GeometricObjects` - 几何对象

---

### 文件 ②: 接口声明 (PublicInterfaces/IInterface.h)

**位置**: `Framework.edu/PublicInterfaces/IInterface.h`

**作用**: 定义接口契约

**模板**:
```cpp
#ifndef ICalculator_h
#define ICalculator_h

#include "CATBaseUnknown.h"

// IID extern declaration (generated by generate_guid_ai.bat)
extern IID IID_ICalculator;

class ICalculator : public CATBaseUnknown
{
    CATDeclareInterface;
    
public:
    virtual HRESULT Add(int a, int b, int* result) = 0;
    virtual HRESULT Subtract(int a, int b, int* result) = 0;
};

#endif
```

**关键点**:
- ✅ 继承 `CATBaseUnknown`
- ✅ 使用 `CATDeclareInterface` 宏
- ✅ 所有方法返回 `HRESULT`
- ✅ 所有方法纯虚函数 (`= 0`)
- ✅ 包含 `extern IID` 声明

---

### 文件 ③: 接口实现 (src/IInterface.cpp)

**位置**: `Framework.edu/Module.m/src/IInterface.cpp`

**作用**: 定义 IID 和 MetaObject

**模板**:
```cpp
#include "ICalculator.h"

// IID definition (generated by generate_guid_ai.bat)
IID IID_ICalculator = {
    0xA1B2C3D4, 0xE5F6, 0x7890,
    { 0xAB, 0xCD, 0xEF, 0x12, 0x34, 0x56, 0x78, 0x90 }
};

// Implement interface metadata
CATImplementInterface(ICalculator, CATBaseUnknown);
```

**为什么需要**:
- 定义唯一的接口 ID (IID)
- 创建 MetaObject 用于类型查询
- 缺少此文件 → `MetaObject unresolved` 链接错误

### 文件 ④: 组件声明 (LocalInterfaces/Component.h)

**位置**: `Framework.edu/Module.m/LocalInterfaces/Component.h`

**作用**: 声明组件类

**模板**:
```cpp
#ifndef Calculator_h
#define Calculator_h

#include "CATBaseUnknown.h"
#include "ICalculator.h"

class Calculator : public CATBaseUnknown
{
    CATDeclareClass;
    
public:
    Calculator();
    virtual ~Calculator();
    
    // ICalculator interface implementation
    virtual HRESULT Add(int a, int b, int* result);
    virtual HRESULT Subtract(int a, int b, int* result);
    
private:
    // Copy constructor and assignment operator (private, unimplemented)
    Calculator(const Calculator&);
    Calculator& operator=(const Calculator&);
};

#endif
```

**关键点**:
- ✅ 使用 `CATDeclareClass` 宏
- ✅ 复制构造和赋值操作符私有化
- ✅ 虚析构函数

### 文件 ⑤: 构建配置 (Imakefile.mk)

**位置**: `Framework.edu/Module.m/Imakefile.mk`

**作用**: 配置编译和链接

**模板**:
```makefile
BUILT_OBJECT_TYPE=LOAD_MODULE

LINK_WITH=JS0GROUP

LOCAL_CCFLAGS=

LOCAL_LNKFLAGS=
```

**语法规则**:
- ✅ `LINK_WITH=` (等号前无空格)
- ❌ `LINK_WITH =` (错误 - 有空格)

**常用链接库**:
- `JS0GROUP` - 核心系统（必需）
- `JS0FM` - 文件管理
- `Mathematics` - 数学库
- `GeometricObjects` - 几何库

### 文件 ⑥: 组件实现 (src/Component.cpp)

**位置**: `Framework.edu/Module.m/src/Component.cpp`

**作用**: 实现组件逻辑

**模板**:
```cpp
#include "Calculator.h"
#include <iostream.h>

// Implement class metadata
CATImplementClass(Calculator, Implementation, CATBaseUnknown, CATNull);

// Bind interface to component (BOA pattern for B28)
CATImplementBOA(ICalculator, Calculator);

// Constructor
Calculator::Calculator()
{
    cout << "Calculator created" << endl;
}

// Destructor
Calculator::~Calculator()
{
    cout << "Calculator destroyed" << endl;
}

// Implement ICalculator::Add
HRESULT Calculator::Add(int a, int b, int* result)
{
    if (result == NULL) return E_FAIL;
    *result = a + b;
    return S_OK;
}

// Implement ICalculator::Subtract
HRESULT Calculator::Subtract(int a, int b, int* result)
{
    if (result == NULL) return E_FAIL;
    *result = a - b;
    return S_OK;
}
```

**关键点**:
- ✅ `CATImplementClass` - 定义类元数据
- ✅ `CATImplementBOA` - 绑定接口和组件（B28 模式）
- ✅ 使用 `iostream.h` 和 `cout`（不是 `<iostream>` 和 `std::cout`）
- ✅ 检查空指针
- ✅ 返回 `S_OK` (成功) 或 `E_FAIL` (失败)

---

### 文件 ⑦: 字典文件 (CNext/code/dictionary/Framework.edu.dico)

**位置**: `Framework.edu/CNext/code/dictionary/Framework.edu.dico`

**作用**: 注册组件和接口映射

**格式**: 3 列，Tab 分隔

**模板**:
```
Calculator	CATImplements	ICalculator
```

**详细格式**:
```
<ComponentName>\t<关键字>\t<InterfaceName>
```

**关键字**:
- `CATImplements` - 组件实现接口
- `CATExtends` - 组件扩展另一个组件

**多接口示例**:
```
Calculator	CATImplements	ICalculator
Calculator	CATImplements	IPersistent
```

**关键点**:
- ✅ 使用 Tab 分隔（不是空格）
- ✅ 文件名必须是 `<FrameworkName>.dico`
- ❌ 缺少此文件 → 组件无法实例化

## CAA 核心库使用

### 集合类

**不要创建自定义集合！使用 CAA 提供的：**

```cpp
// 整数列表
#include "CATListOfInt.h"
CATListOfInt intList;
intList.Append(10);
intList.Append(20);
int value = intList[1];  // 10 (1-based index)

// 浮点列表
#include "CATListOfDouble.h"
CATListOfDouble doubleList;

// 指针列表
#include "CATListOfCATUnicodeString.h"
CATLISTP(CATUnicodeString) stringList;
```

### 字符串

**不要使用 char* 或 std::string！使用：**

```cpp
#include "CATUnicodeString.h"

CATUnicodeString str1("Hello");
CATUnicodeString str2 = " World";
CATUnicodeString result = str1 + str2;

// 转换
const char* cstr = result.ConvertToChar();
int length = result.GetLengthInChar();
```

### 数学

**不要创建自定义向量/矩阵！使用：**

```cpp
#include "CATMathPoint.h"
#include "CATMathVector.h"
#include "CATMathTransformation.h"

CATMathPoint p1(1.0, 2.0, 3.0);
CATMathVector v1(0.0, 0.0, 1.0);

double distance = p1.DistanceTo(p2);
CATMathVector cross = v1 * v2;  // Cross product
```

## 内存管理规则

### 规则 10: QueryInterface 后必须 Release

```cpp
IInterface* pInterface = NULL;
HRESULT hr = component->QueryInterface(IID_IInterface, (void**)&pInterface);

if (SUCCEEDED(hr) && pInterface) {
    // 使用接口
    pInterface->DoSomething();
    
    // 必须释放
    pInterface->Release();
    pInterface = NULL;
}
```

**原因**: `QueryInterface` 增加引用计数，不释放会内存泄漏。

### 规则 11: 存储指针时必须 AddRef

```cpp
class MyClass {
    IInterface* _storedPtr;
    
public:
    MyClass() : _storedPtr(NULL) {}
    
    ~MyClass() {
        if (_storedPtr) {
            _storedPtr->Release();
            _storedPtr = NULL;
        }
    }
    
    void SetInterface(IInterface* p) {
        if (_storedPtr) {
            _storedPtr->Release();
        }
        
        _storedPtr = p;
        
        if (_storedPtr) {
            _storedPtr->AddRef();  // 必须增加引用计数
        }
    }
};
```

**原因**: 存储指针意味着延长对象生命周期，必须增加引用计数。

## 组件实例化

### 创建组件实例

**错误方式**:
```cpp
Calculator* calc = new Calculator();  // ❌ 不要这样做！
```

**正确方式**:
```cpp
ICalculator* pCalc = NULL;
HRESULT hr = ::CATInstantiateComponent(
    "Calculator",              // 组件名称（字典文件中的）
    IID_ICalculator,           // 接口 ID
    (void**)&pCalc             // 输出接口指针
);

if (SUCCEEDED(hr) && pCalc) {
    int result = 0;
    pCalc->Add(10, 20, &result);
    cout << "Result: " << result << endl;
    
    pCalc->Release();  // 必须释放
    pCalc = NULL;
}
```

## 扩展已有组件

### 为已有 CATIA 组件添加接口

**IMyExtension.h**:
```cpp
class IMyExtension : public CATBaseUnknown {
    CATDeclareInterface;
public:
    virtual HRESULT MyNewMethod() = 0;
};
```

**MyExtension.h**:
```cpp
class MyExtension : public CATBaseUnknown {
    CATDeclareClass;
public:
    virtual HRESULT MyNewMethod();
};
```

**MyExtension.cpp**:
```cpp
CATImplementClass(MyExtension, DataExtension, CATBaseUnknown, CATICsoGeometry);
//                              ^^^^^^^^^^^^^ 使用 DataExtension

CATImplementBOA(IMyExtension, MyExtension);
```

**Dictionary**:
```
CATICsoGeometry	IMyExtension	libMyModule
```

**关键点**:
- 使用 `DataExtension` 作为第二个参数
- 第四个参数是要扩展的组件类
- 字典格式不同：`<TargetComponent>\t<NewInterface>\t<ModuleName>`

## 编译命令详解

### 基本命令

```cmd
# 编译单个模块
mkmk -u Module.m

# 编译所有模块
mkmk -au

# 带调试信息编译
mkmk -g -u Module.m

# 清理后重新编译
mkmk -a Module.m

# 详细输出
mkmk -u -v Module.m
```

### 环境初始化

```cmd
# 方式 1: 使用 TCK
<CATIA_INSTALL>\intel_a\code\bin\tck_init.bat
tck_profile <ProfileName>

# 方式 2: 手动设置（不推荐）
set PATH=%PATH%;<CATIA_INSTALL>\intel_a\code\bin
```

### 常见编译错误

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| `C1083: Cannot open include file` | 头文件路径错误 | 检查 Imakefile.mk 的 LINK_WITH |
| `LNK2019: unresolved external symbol` | 缺少库链接 | 添加缺少的库到 LINK_WITH |
| `MetaObject unresolved` | 缺少 Interface.cpp | 创建 IInterface.cpp |
| `mkmk not found` | 环境未初始化 | 运行 tck_init |

## 运行时测试

### 在 CATIA 中测试组件

**创建测试宏** (CATScript):
```vb
Sub CATMain()
    Dim calc As ICalculator
    Set calc = CATIA.GetItem("Calculator")
    
    If Not calc Is Nothing Then
        Dim result As Integer
        calc.Add 10, 20, result
        MsgBox "10 + 20 = " & result
    Else
        MsgBox "Failed to create Calculator"
    End If
End Sub
```

**C++ 测试代码**:
```cpp
#include "ICalculator.h"

void TestCalculator() {
    ICalculator* pCalc = NULL;
    HRESULT hr = ::CATInstantiateComponent(
        "Calculator",
        IID_ICalculator,
        (void**)&pCalc
    );
    
    if (SUCCEEDED(hr) && pCalc) {
        int result = 0;
        
        // 测试 Add
        hr = pCalc->Add(10, 20, &result);
        if (SUCCEEDED(hr)) {
            cout << "Add test passed: 10 + 20 = " << result << endl;
        }
        
        // 测试 Subtract
        hr = pCalc->Subtract(30, 15, &result);
        if (SUCCEEDED(hr)) {
            cout << "Subtract test passed: 30 - 15 = " << result << endl;
        }
        
        pCalc->Release();
    } else {
        cout << "Failed to instantiate Calculator" << endl;
    }
}
```
