## 快速开始

### 环境要求

- CATIA V5 R28+ / 3DEXPERIENCE R2018x+
- CAA RADE 或 Build Time 环境
- Python 3.7+ (CADE)

### 编译

```bash
cd scripts
python build.py
```

### 安装

```bash
cd scripts
python install.py
```
